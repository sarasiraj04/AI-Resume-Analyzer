# 📄 AI Resume Analyzer using Gemini AI

An AI-powered Resume Analyzer that uses Google Gemini API to analyze resumes, calculate ATS scores, compare resumes with job descriptions, and provide improvement suggestions.

## 🚀 Features

- 📄 Upload Resume PDF
- 🔍 Extract text from PDF automatically
- 🤖 AI-based Resume Analysis using Gemini
- 📊 ATS Score Prediction
- 💼 Resume vs Job Description Matching
- ✅ Identify matching skills
- ❌ Find missing skills
- 💡 Get resume improvement suggestions
